# apple-action/prepare-xcode

Select Xcode version.

## Behavior

1. Select specific version of Xcode.

## Usage

```
- uses: ab180/apple-action/prepare-xcode@v2
  with:
    version: 15.3
```

Required    | Name                                          | Type      | Description
---         | ---                                           | ---       | ---
O           | version                                       | String    | Version of Xcode
